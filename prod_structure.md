### Key Directories

* **`/src/total_bankroll/`**: The main application package containing all core logic, models, routes, and templates.
* **`/migrations/`**: Contains database migration scripts generated by Flask-Migrate (Alembic).
* **`/tests/`**: Holds all the tests for the application.
* **`/static/`**: Stores static assets like CSS, JavaScript, and images.
* **`/templates/`**: Contains all Jinja2 HTML templates for the frontend.
* **`/home/pythonpydev/.virtualenvs/bankroll_venv`**: Python virtual environment
